Introducción Básica a Bases de Datos

Las bases de datos son un conjunto de datos ordenados, procesados y almacenados para su fácil localización. Los datos cuando se compilan y se hacen entendibles se vuelven de vital importancia. Pues esto nos permite tomar decisiones.
